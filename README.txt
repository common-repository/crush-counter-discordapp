== Installation ==

=PORTUGUÊS=

NO DISCORDAPP

1. Vá em Server Settings->Webhooks
2. Crie um novo Webhook e copie a URL.

NO WORDPRESS

1. Vá em Configurações->CCN DiscordApp
2. Cole a URL do Webhook no campo WEBHOOK.
3. Digite um nome para o Bot (Optional)
4. Escolhe uma nova imagem para o Bot(Optional).
5. Salve

=ENGLISH=

ON DISCORDAPP

1. Go to Server Settings->Webhooks
2. Create a new Webhook and Copy URL.

ON WORDPRESS

1. Go to Settings->CCN DiscordApp
2. Paste the WebHook URL on WebHook URL Field.
3. Enter a Bot Name (Optional)
4. Choose a Bot Avatar (Optional).
5. Save
